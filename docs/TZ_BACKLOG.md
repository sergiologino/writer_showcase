## Частично / упрощённо относительно ТЗ

| Требование ТЗ | Как сейчас | Дальше |
|---------------|------------|--------|
| Чекбоксы каналов у поста, расписание, статусы pending/scheduled/… | Публикация при первом `PUBLISHED`+`PUBLIC`, без привязки чекбоксов к посту | Модель `PostChannelBinding`, UI выбора каналов, расписание |
| Повторная публикация, внешние URL по каналам | Ретраи доставки; нет republish/rebind UI | API + UI republish, хранение `external_url` |
| Варианты текста на канал (перевод, сокращение) | Нет `PostVariant` | Таблицы + AI + редактор варианта |
| Prompt templates: версии, откат, preview, параметры модели | Есть тексты промптов; без версионирования и preview | Как в §3.7 ТЗ |
| Аналитика: просмотры, лайки, polling метрик каналов | Нет | `AnalyticsSnapshot`, планировщик, UI |
| RSS/Atom | Нет | `GET /api/public/rss` + ссылки |
| Open Graph / SEO | Минимально | Метатеги на публичных страницах |
| Роли owner/admin/editor/viewer, RBAC | Упрощённо | Проверки по ролям в API |
| Шифрование секретов каналов at rest | Нет | KMS/шифрование полей |
| Медиа по URL / AI-картинки | Нет | Эндпоинты + UI |
| Лендинг «эпичный» | Нет отдельной страницы | Маркетинговый `/` или `/landing` |
| Инфографика, дашборд аналитики | Нет | Recharts, раздел `/app/analytics` |
| X, Facebook, Reddit, vc.ru, Habr | Нет | Адаптеры по очереди |
| Мессенджер МАКС (MAX) | Нет | Запланировано: см. [`docs/ai/TODO.md`](./ai/TODO.md) |
| Аудит, Prometheus | Нет | По мере зрелости |

## Рекомендуемый порядок следующих итераций

1. **Продуктовая ясность (короткий цикл)**  
   Подсказки в UI (черновик vs публичный блог), ссылки на публичный блог из шапки — уже усилены; при необходимости — дашборд «статусы каналов» у поста.

2. **Публикация как в ТЗ**  
   Привязка поста к каналам (чекбоксы), отображение статуса/URL по каналу, повторная публикация.

3. **Расписание**  
   `scheduled_at`, отложенная публикация, календарь (`/app/calendar`).

4. **Варианты контента на канал**  
   `PostVariant`, перевод/сокращение через существующий AI.

5. **Аналитика своего сайта + polling внешних метрик**  
   Просмотры публичных страниц, снимки по каналам.

6. **RSS, OG, роли, шифрование секретов**  
   Закрытие MVP1 из §16 ТЗ по мере приоритета.
