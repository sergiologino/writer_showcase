# Writer Showcase / Altacod Publisher — обзор

## Назначение

Многопользовательская платформа для авторов и команд по модели **POSSE** (Publish Own Site, Syndicate Elsewhere): контент создаётся и хранится как первоисточник в системе, затем выборочно синдицируется во внешние каналы (соцсети, мессенджеры, блог-площадки).

## Продукт в одном абзаце

Редактор и хранилище публикаций, публичный блог, интеграции с каналами (публикация, расписание, повторы), AI-помощник через настраиваемые prompt templates, варианты контента под каналы, аналитика (внутренняя и по каналам), workspace и роли.

## Источник требований

Нормативное ТЗ и blueprint: `E:\1_MyProjects\!_POSSE\altacod_publisher_spec.md` (в репозитории при необходимости можно добавить копию или ссылку в README — по отдельному решению).

## Кодовое имя репозитория

`writer_showcase` — рабочее имя проекта в файловой системе; продукт в спецификации назван **Altacod Publisher**.

Реализация MVP ведётся в `apps/backend` и `apps/web` (см. `CURRENT_STATE.md`).

## Этапы (из ТЗ)

- **MVP 1**: auth, workspace, посты, категории/теги, медиа, публичный блог, Telegram + VK, базовый AI и UI промптов, история публикаций, аналитика своего сайта, базовый polling метрик, темы UI.
- **MVP 2–3**: расширение каналов, расписание, варианты по каналам, расширенная аналитика и пр. — по `altacod_publisher_spec.md`.
